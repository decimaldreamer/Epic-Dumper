
# Epic Dumper
> Dump native and .NET assemblies
## Features
-  Login and Register System powered by AUTH.GG.
- Virtual Memory
- HeapView
- Enum Appdomains
- .NET Performance
- JIT Hook Detection
- Enviroment Variables
- Directories List
- Generating Dump

## Setup
Enter the credentials you got from AUTH.GG in the **program.cs** field.
